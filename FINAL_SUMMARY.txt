## 💡 EXAMPLE USAGE

### Create a Conversation
```bash
curl -X POST http://localhost:8000/api/v1/agent/conversation \
  -H "Content-Type: application/json" \
  -d '{"user_id":"you","title":"My Chat"}'
```

### Send a Message
```bash
curl -X POST http://localhost:8000/api/v1/agent/message \
  -H "Content-Type: application/json" \
  -d '{
    "conversation_id":"<conv-id>",
    "content":"What is 5 plus 3?",
    "user_id":"you"
  }'
```

Response:
```json
{
  "response": "The result of 5 add 3 is 8",
  "tool_used": "calculator",
  "reasoning": "Used calculator tool to answer your question"
}
```

# 🎯 QUICK START - Run Frontend & Backend

## 3-Step Quick Start (5 minutes)

### Step 1: Setup (One Time)
```bash
cd E:\Vibe-Coding-AI\Ai-Agent\frontend
npm install
```

### Step 2: Terminal 1 - Start Backend
```bash
cd E:\Vibe-Coding-AI\Ai-Agent
python startup.py
```

### Step 3: Terminal 2 - Start Frontend
```bash
cd E:\Vibe-Coding-AI\Ai-Agent\frontend
npm start
```

**Open Browser:** http://localhost:3000

## 🔌 Connected Endpoints

Frontend automatically connects to these backend endpoints:

```
✅ GET  /api/v1/health
✅ GET  /api/v1/agent/tools
✅ POST /api/v1/agent/conversation
✅ GET  /api/v1/agent/conversations/{user_id}
✅ POST /api/v1/agent/message
✅ GET  /api/v1/agent/conversation/{id}/history
✅ POST /api/v1/agent/tools/{name}/execute
```

---

## 🚨 If Connection Fails

1. Make sure backend is running: `python startup.py`
2. Check backend is on port 8000
3. Check frontend is on port 3000
4. Press F12 to see browser console errors
5. Click "Retry Connection" button in app

## COMMON ERRORS & FIXES

**Error: "npm: command not found"**
→ Install Node.js from https://nodejs.org/

**Error: "Cannot connect to backend"**
→ Make sure Terminal 1 is running `python startup.py`

**Error: "Port 3000 already in use"**
→ Run this in Terminal 2 instead:
```bash
cd E:\Vibe-Coding-AI\Ai-Agent\frontend && PORT=3001 npm start
```

**Error: No messages appearing**
→ Press F12 to open browser console, look for errors
